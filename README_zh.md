## 介绍

`VisitOnce`是一个阅后即焚的PHP Web程序

## 如何运行

1. 下载源码并解压到`www`文件夹或者任何你想要的位置
2. 修改文件属主
3. 配置`apache2`，让其加载`.htaccess`
4. 把内容放到`contents`文件夹并测试
